## Key Insight

**No CDP product globally has a true conversational AI agent.**
Salesforce Agentforce is the sole exception but:
- Locked into Salesforce ecosystem (not independent product)
- Priced at $2 per conversation
- CRM-focused, weak CDP audience segmentation
- Unavailable in China

## CDP Core Feature Modules (Standard)

1. Data integration — multi-source collection (SDK/API/file import)
2. OneID / Unified profiles — cross-device/channel ID resolution
3. Tag system — rule-based, computed, predictive tags
4. Audience segmentation — visual condition combos, behavioral sequences, lookalike
5. Marketing canvas — drag-and-drop flow, triggers, multi-channel, A/B testing
6. Analytics — event/funnel/retention/attribution analysis
7. Data activation — sync to ad platforms, webhook, messaging channels
